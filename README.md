___


![preview_nova](dist/img/preview_nova.png)

### Верска выполнена с использование следующих технологий:

* Gulp
* SCSS
* SlickSliders
* FontAwesome 5
___

### Дополнительно

* Реализованы эффекты на странице
* Добавлен адаптив страницы
* Добавлено адаптивное меню под мобильные устройства
